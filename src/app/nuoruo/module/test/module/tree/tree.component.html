tree 组件 nr-tree

只提供了一个tree骨架。用户需在在插槽（内容投影）中自定义展示的条目数据。item 当前展示的条目数据， index 根据下标计算的值, level 当前层级

<nr-tree [(ngModel)]="treeData">
  <ng-template nrTree let-item let-index="index" let-level="level">
    自定义内容
  </ng-template>
</nr-tree>
下面是效果展示
{{ item.flagExpand ? "关闭子项" : "展开子项" }} {{ "当前绑定的值" + item.value + " -当前显示的值 " + item.v + " - 定标：" + index + " - 层级：" + level }} 当前状态： {{ item.flagExpand ? "展开" : "关闭" }}
